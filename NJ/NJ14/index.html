A JavaScript Runtime Environment (JRE) is an environment that provides the necessary infrastructure and APIs for executing JavaScript code.

Components of a JavaScript Runtime Environment :
1) JavaScript Engine : The core engine that parses, interprets, and executes JavaScript code.
2) Event Loop : A mechanism that handles asynchronous operations and ensures non-blocking execution.
3) APIs : Provide functionalities beyond the core JavaScript language, such as: Browser-specific APIs: DOM, Fetch, Web APIs.
4) Microtask and Macrotask Queues : Microtasks Queue: High-priority tasks (e.g., Promises). Callback Queue: Lower-priority tasks (e.g., setTimeout).
